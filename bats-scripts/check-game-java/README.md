# check-game-java.bat

## 📋 Overview

Check-game-java would Read everything out of the currently running game JVM, including Java version, installation path, java flags, anything that could be potentially useful.

This was important for diagnostic purposes as it allowed me to get the full system information, and confirm if the correct JVM flags were set by the application, what version the user was running and where the instillation was, as Finding game instlations wasn't straight forward for some users due to game installing under Appdata/Roaming, or Appdata/Local among other factors.
